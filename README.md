# QonciliaFront

## Requisitos
- Node.js 18+

## Ejecutar
1. `npm install`
2. Copiar `.env.example` a `.env` y ajustar `VITE_API_URL`
3. `npm run dev`

## Pantallas
- `/login`
- `/register`
- `/` (home, protegida)
- `/users` (solo admin/superadmin)
